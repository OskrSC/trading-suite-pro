# Trading Suite Pro 📈

> **Plataforma integral de análisis de mercados financieros**

---

## 📌 Descripción

**Trading Suite Pro** es una plataforma de código abierto desarrollada en **Python** para el **análisis técnico, backtesting y visualización de mercados financieros** (acciones, criptomonedas, etc.). Utiliza **Streamlit** para ofrecer una interfaz web interactiva y fácil de usar, ideal para traders, analistas y desarrolladores que buscan automatizar sus estrategias de trading.

---

## ✨ Características

- **Análisis Técnico**: Herramientas para analizar gráficos, indicadores técnicos y patrones de mercado.
- **Backtesting**: Pruebas de estrategias de trading con datos históricos para evaluar su rendimiento.
- **Dashboard Interactivo**: Visualización de datos en tiempo real con **Streamlit**.
- **Módulos Especializados**:
  - `dashboard_tecnico.py`: Análisis técnico avanzado.
  - `backtesting.py`: Simulación de estrategias.
  - `analisis_sentimiento.py`: Análisis de sentimiento de mercado.
- **Extensible**: Fácil de personalizar y ampliar con nuevos módulos.
